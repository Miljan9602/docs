# RFQ Protocol Documentation

Documentation for the RFQ (Request for Quote) Protocol built with [Mintlify](https://mintlify.com).

## Local Development

```bash
# Install dependencies
npm install

# Start local preview
npm run dev
```

Documentation will be available at `http://localhost:3000`.

## Deployment

This documentation is deployed via Mintlify. Connect this repository to your Mintlify dashboard for automatic deployments.
