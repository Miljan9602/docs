---
title: "Get Quote"
api: "GET /api/v1/quote"
---

Get the best available quote for a specific token swap.

## Query Parameters

<ParamField query="input_token" type="string" required>
  The token address to sell (0x-prefixed, 40 hex characters)
</ParamField>

<ParamField query="output_token" type="string" required>
  The token address to buy (0x-prefixed, 40 hex characters)
</ParamField>

<ParamField query="amount" type="string" required>
  Amount to swap in wei (as a numeric string). For a token with 6 decimals, `1000000` = 1 token.
</ParamField>

<ParamField query="chain_id" type="string" required>
  The blockchain chain ID for a supported network
</ParamField>

## Response

<ResponseField name="status" type="string">
  Response status: `"ok"` on success, `"error"` on failure
</ResponseField>

<ResponseField name="quote_id" type="string">
  Unique quote identifier (UUID format)
</ResponseField>

<ResponseField name="input_token" type="string">
  The input token address (lowercase)
</ResponseField>

<ResponseField name="output_token" type="string">
  The output token address (lowercase)
</ResponseField>

<ResponseField name="amount_in" type="string">
  Total amount of input tokens required (in wei). This includes the protocol fee.
</ResponseField>

<ResponseField name="amount_out" type="string">
  Amount of output tokens you'll receive (in wei)
</ResponseField>

<ResponseField name="fee" type="string">
  Protocol fee amount (in wei of input token)
</ResponseField>

<ResponseField name="expiry" type="number">
  Unix timestamp (milliseconds) when the quote expires.
</ResponseField>

<ResponseField name="chain_id" type="string">
  The chain ID
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://dev-rfq.saphyre.xyz/api/v1/quote?\
input_token=<INPUT_TOKEN>&\
output_token=<OUTPUT_TOKEN>&\
amount=<AMOUNT>&\
chain_id=<CHAIN_ID>"
```

```javascript JavaScript
const params = new URLSearchParams({
  input_token: INPUT_TOKEN,
  output_token: OUTPUT_TOKEN,
  amount: AMOUNT,  // In token's smallest unit (e.g., wei)
  chain_id: CHAIN_ID,
});

const response = await fetch(
  `https://dev-rfq.saphyre.xyz/api/v1/quote?${params}`
);
const data = await response.json();

if (data.status === "ok") {
  console.log("Quote ID:", data.quote_id);
  console.log("You'll receive:", data.amount_out);
  console.log("Fee:", data.fee);
  console.log("Expires:", new Date(data.expiry));
}
```

```python Python
import requests

response = requests.get(
    "https://dev-rfq.saphyre.xyz/api/v1/quote",
    params={
        "input_token": INPUT_TOKEN,
        "output_token": OUTPUT_TOKEN,
        "amount": AMOUNT,  # In token's smallest unit
        "chain_id": CHAIN_ID
    }
)
data = response.json()
print(f"Quote: {data['amount_out']} output tokens, Fee: {data['fee']}")
```
</RequestExample>

## Notes

- The quote TTL is configurable per market (default 15 seconds)
- The `quote_id` is required when submitting an intent
- Always check the `expiry` field before submitting - expired quotes will be rejected
- Amount values are always in wei (base units of the token)

<Warning>
  Quote prices may vary based on market conditions. Always display the final `amount_out` to users before they sign.
</Warning>
