The RFQ Protocol API allows frontends and integrators to request quotes and execute swaps.

## Base URLs

| Environment | URL |
|-------------|-----|
| Production | `https://api-rfq.dragonswap.app` |
| Testnet | `https://testnet-api-rfq.dragonswap.app` |

## Authentication

The API is public and does not require authentication.

## API Version

| Version | Status | Features |
|---------|--------|----------|
| **v1** | Live | REST API, full fills only |

All endpoints are prefixed with `/api/v1/`.

## Request Format

- All requests use JSON
- POST requests require `Content-Type: application/json`
- Query parameters for GET requests

## Response Format

All responses follow this structure:

```json
// Success
{
  "status": "ok",
  "data": { ... }
}

// Error
{
  "status": "error",
  "error": "Error message" | { "field": ["Error"] }
}
```

## Rate Limits

| Endpoint | Limit |
|----------|-------|
| `/quote` | 100 requests/minute |
| `/markets` | 60 requests/minute |
| `/intent` | 30 requests/minute |

<Note>
  Contact the team if you need higher rate limits for production use.
</Note>

## Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/v1/config` | Get chain configuration |
| GET | `/api/v1/markets` | List available trading pairs |
| GET | `/api/v1/quote` | Get best price quote |
| POST | `/api/v1/intent` | Execute a swap |
| GET | `/api/v1/health` | Check server status |

## Quick Start

### 1. Get Available Markets

```bash
curl "https://api-rfq.dragonswap.app/api/v1/markets?chain_id=<CHAIN_ID>"
```

### 2. Request a Quote

```bash
curl "https://api-rfq.dragonswap.app/api/v1/quote?\
input_token=<INPUT_TOKEN>&\
output_token=<OUTPUT_TOKEN>&\
amount=<AMOUNT>&\
chain_id=<CHAIN_ID>"
```

### 3. Submit Intent

```bash
curl -X POST "https://api-rfq.dragonswap.app/api/v1/intent" \
  -H "Content-Type: application/json" \
  -d '{
    "quote_id": "rfq-q-...",
    "chain_id": "<CHAIN_ID>",
    "user_address": "0x...",
    "swap_intent": { ... },
    "signature_params": { ... }
  }'
```
