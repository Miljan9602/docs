---
title: "Get Markets"
api: "GET /api/v1/markets"
---

Get the list of available trading pairs.

## Query Parameters

<ParamField query="chain_id" type="string" required>
  The blockchain chain ID for a supported network
</ParamField>

## Response

<ResponseField name="status" type="string">
  Response status: `"ok"` on success, `"error"` on failure
</ResponseField>

<ResponseField name="markets" type="array">
  List of available trading pairs
  <Expandable title="Market object">
    <ResponseField name="input_token" type="string">
      The input token address (0x-prefixed, lowercase)
    </ResponseField>
    <ResponseField name="output_token" type="string">
      The output token address (0x-prefixed, lowercase)
    </ResponseField>
    <ResponseField name="min_size" type="number | null">
      Minimum order size in **Input Token** units. `null` means no minimum.
    </ResponseField>
    <ResponseField name="max_size" type="number | null">
      Maximum order size in **Input Token** units. Dynamically derived from solver on-chain balances. `null` means no maximum.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://dev-rfq.saphyre.xyz/api/v1/markets?chain_id=<CHAIN_ID>"
```

```javascript JavaScript
const CHAIN_ID = "..."; // Your chain ID

const response = await fetch(
  `https://dev-rfq.saphyre.xyz/api/v1/markets?chain_id=${CHAIN_ID}`
);
const data = await response.json();
console.log(data.markets);
```

```python Python
import requests

CHAIN_ID = "..."  # Your chain ID

response = requests.get(
    "https://dev-rfq.saphyre.xyz/api/v1/markets",
    params={"chain_id": CHAIN_ID}
)
data = response.json()
print(data["markets"])
```
</RequestExample>

## Notes

- Markets always appear as directional pairs (A→B and B→A are separate entries with their own `max_size`)
- `max_size` reflects each solver's available on-chain balance, converted to input token units
- Each direction may have a different `max_size` depending on solver liquidity
