---
title: "Get Config"
api: "GET /api/v1/config"
---

Get chain configuration including contract addresses and TTL settings.

## Query Parameters

<ParamField query="chain_id" type="string" required>
  The blockchain chain ID for a supported network
</ParamField>

## Response

<ResponseField name="status" type="string">
  Response status: `"ok"` on success, `"error"` on failure
</ResponseField>

<ResponseField name="chain_id" type="string">
  The chain ID
</ResponseField>

<ResponseField name="chain_name" type="string">
  Human-readable chain name
</ResponseField>

<ResponseField name="rfq_settlement_address" type="string">
  RFQSettlement contract address for this chain
</ResponseField>

<ResponseField name="permit2_address" type="string">
  Permit2 contract address for this chain
</ResponseField>

<ResponseField name="min_quote_ttl_ms" type="number">
  Minimum quote TTL in milliseconds
</ResponseField>

<ResponseField name="max_quote_ttl_ms" type="number">
  Maximum quote TTL in milliseconds
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://dev-rfq.saphyre.xyz/api/v1/config?chain_id=<CHAIN_ID>"
```

```javascript JavaScript
const CHAIN_ID = "..."; // Your chain ID

const response = await fetch(
  `https://dev-rfq.saphyre.xyz/api/v1/config?chain_id=${CHAIN_ID}`
);
const data = await response.json();

if (data.status === "ok") {
  console.log("Settlement:", data.rfq_settlement_address);
  console.log("Permit2:", data.permit2_address);
}
```

```python Python
import requests

CHAIN_ID = "..."  # Your chain ID

response = requests.get(
    "https://dev-rfq.saphyre.xyz/api/v1/config",
    params={"chain_id": CHAIN_ID}
)
data = response.json()
print(f"Settlement: {data['rfq_settlement_address']}")
print(f"Permit2: {data['permit2_address']}")
```
</RequestExample>

## Notes

- Use this endpoint to fetch contract addresses dynamically instead of hardcoding them
- TTL values indicate the expected quote validity range for this chain
- Supported chains may be expanded in the future
