## User Pools

```
GET https://sei-api.dragonswap.app/api/v1/user/{address}/pools
```

Returns a user's liquidity pool positions across V2 and V3 pools, including total liquidity in USD.

### Path Parameters

<ParamField path="address" type="string" required>
  User wallet address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="tokens" type="array">
  Tokens in the user's pools. Each contains:
  - `address` (string) — Token contract address
  - `name` (string) — Token name
  - `symbol` (string) — Token symbol
  - `usd_price` (number) — Current USD price
  - `decimals` (number) — Token decimals
</ResponseField>

<ResponseField name="pools" type="array">
  User's pool positions. Each contains the standard pool fields (`pool_address`, `token0_address`, `token1_address`, `liquidity`, `type`, etc.) plus user-specific position data.
</ResponseField>

<ResponseField name="incentives" type="array">
  Active incentives for the user's pools.
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/user/{address}/pools"
```

```javascript JavaScript
const response = await fetch(
  `https://sei-api.dragonswap.app/api/v1/user/${walletAddress}/pools`
);
const data = await response.json();

console.log(`User has ${data.pools.length} pool positions`);
```
</RequestExample>

## User Portfolio

```
GET https://sei-api.dragonswap.app/api/v1/user/{address}/portfolio
```

Returns a user's portfolio overview including historical value, token holdings, and total USD value.

### Path Parameters

<ParamField path="address" type="string" required>
  User wallet address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="meta" type="object">
  Portfolio metadata:
  - `native_token_usd_price` (number) — Current SEI price in USD
  - `change` (object) — Native token price changes:
    - `hourly` (number) — Last hour change %
    - `daily` (number) — Last 24h change %
    - `weekly` (number) — Last 7d change %
    - `monthly` (number) — Last 30d change %
</ResponseField>

<ResponseField name="portfolio_history" type="array">
  Portfolio value history over time. Each entry contains:
  - `timestamp` (number) — Unix timestamp
  - `liquidity_usd` (number) — Liquidity value in USD
  - `farm_balance_usd` (number) — Farm balance value in USD
</ResponseField>

<ResponseField name="tokens" type="array">
  User's token holdings. Each contains:
  - `address` (string) — Token contract address
  - `name` (string) — Token name
  - `symbol` (string) — Token symbol
  - `usd_price` (number) — Current USD price
  - `decimals` (number) — Token decimals
  - `change` (object) — Price changes (`hourly`, `daily`, `weekly`, `monthly`)
  - `user_balance` (number) — User's token balance
</ResponseField>

<ResponseField name="total_usd_value" type="number">
  Total portfolio value in USD.
</ResponseField>

<ResponseField name="native_token" type="object">
  Native SEI token details:
  - `balance` (number) — User's native SEI balance
  - `usd_price` (number) — Current SEI price in USD
</ResponseField>

## User Farms

```
GET https://sei-api.dragonswap.app/api/v1/user/{address}/farms
```

Returns all farms where the user has a non-zero balance.

### Path Parameters

<ParamField path="address" type="string" required>
  User wallet address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="tokens" type="array">
  Tokens across the user's farms.
</ResponseField>

<ResponseField name="pools" type="array">
  Pools associated with the user's farms.
</ResponseField>

<ResponseField name="farms" type="array">
  User's farm positions. Each contains the standard farm fields (`farm_address`, `pooled_token_address`, `reward_token_address`, `apy`, `tvl`, etc.).
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/user/{address}/farms"
```
</RequestExample>

## User Single Farm

```
GET https://sei-api.dragonswap.app/api/v1/user/{address}/farms/{farm}
```

Returns a user's position and history for a specific farm, including total reward and booster claims.

### Path Parameters

<ParamField path="address" type="string" required>
  User wallet address
</ParamField>

<ParamField path="farm" type="string" required>
  Farm contract address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="tokens" type="array">
  Tokens associated with the farm (reward and booster tokens).
</ResponseField>

<ResponseField name="farm" type="object">
  Farm details (same fields as the Get Farm endpoint).
</ResponseField>

<ResponseField name="history" type="array">
  Farm interaction history. Each event contains:
  - `timestamp` (number) — Unix timestamp
  - `user` (string) — User wallet address
  - `type` (string) — `"deposits"`, `"withdraws"`, or `"payouts"`
  - `pooled_token_address` (string) — Pooled token address
  - `block_number` (number) — Block number
  - `farm_address` (string) — Farm contract address
  - `tx_hash` (string) — Transaction hash
  - `amount` (number) — Token amount (for deposits/withdrawals)
  - `reward_token_claim` (number) — Reward tokens claimed (for payouts)
  - `booster_token_claim` (number) — Booster tokens claimed (for payouts)
</ResponseField>

<ResponseField name="payouts" type="object">
  Aggregate payout totals:
  - `reward_token_claims` (number) — Total reward tokens claimed
  - `total_booster_claims` (number) — Total booster tokens claimed
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/user/{address}/farms/{farm}"
```
</RequestExample>

## User Staking

```
GET https://sei-api.dragonswap.app/api/v1/user/{address}/stakings/{staker}
```

Returns a user's staking events for a specific staking contract.

### Path Parameters

<ParamField path="address" type="string" required>
  User wallet address
</ParamField>

<ParamField path="staker" type="string" required>
  Staking contract address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="events" type="array">
  User's staking events (deposits, withdrawals, claims).
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/user/{address}/stakings/{staker}"
```
</RequestExample>
