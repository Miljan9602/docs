## List Farms

```
GET https://sei-api.dragonswap.app/api/v1/farms
```

Returns all active yield farms with their associated tokens and pools.

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="tokens" type="array">
  Tokens used across farms. Each contains:
  - `address` (string) — Token contract address
  - `name` (string) — Token name
  - `symbol` (string) — Token symbol
  - `usd_price` (number) — Current USD price
  - `decimals` (number) — Token decimals
</ResponseField>

<ResponseField name="pools" type="array">
  Pools associated with the farms (same shape as the Pools endpoint).
</ResponseField>

<ResponseField name="farms" type="array">
  Array of farm objects. Each contains:
  - `farm_address` (string) — Farm contract address
  - `pooled_token_address` (string) — Staked token address
  - `reward_token_address` (string) — Reward token address
  - `booster_token_address` (string | null) — Booster token address
  - `apy` (number) — Annual percentage yield
  - `tvl` (number) — Total value locked in USD
  - `type` (string) — Farm type: `"basic"` or `"boosted"`
  - `unlock_rate` (number) — Reward per second
  - `booster_unlock_rate` (number | null) — Booster reward per second
  - `is_lp_pooled_token` (boolean) — Whether staked token is an LP token
  - `end_timestamp` (number) — Farm end timestamp
  - `start_timestamp` (number) — Farm start timestamp
  - `total_rewards_usd` (number) — Total rewards value in USD
  - `campaign` (string | null) — Campaign identifier
  - `is_funded` (boolean | null) — Whether the farm is funded
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/farms"
```

## Farm History

```
GET https://sei-api.dragonswap.app/api/v1/farms/{address}/history
```

Returns the transaction history for a farm (deposits, withdrawals, payouts).

### Path Parameters

<ParamField path="address" type="string" required>
  Farm contract address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="tokens" type="array">
  Tokens associated with this farm (reward and booster tokens).
</ResponseField>

<ResponseField name="history" type="array">
  Array of farm history events. Each contains:
  - `timestamp` (number) — Unix timestamp
  - `amount` (number) — Token amount (for deposits/withdrawals)
  - `user` (string) — User wallet address
  - `type` (string) — Event type: `"deposits"`, `"withdraws"`, or `"payouts"`
  - `pooled_token_address` (string) — Pooled token address
  - `block_number` (number) — Block number
  - `farm_address` (string) — Farm contract address
  - `tx_hash` (string) — Transaction hash

  Payout events additionally include:
  - `reward_token_claim` (number) — Reward tokens claimed
  - `booster_token_claim` (number) — Booster tokens claimed (for boosted farms)
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/farms/{address}/history"
```
</RequestExample>

## Farm Stats

```
GET https://sei-api.dragonswap.app/api/v1/farms/{address}/stats
```

Returns historical statistics for a farm across multiple time periods.

### Path Parameters

<ParamField path="address" type="string" required>
  Farm contract address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="stats" type="object">
  Object containing stat arrays for each time period:
  - `daily` — Last 24 hours
  - `weekly` — Last 7 days (up to 168 records)
  - `monthly` — Last 30 days (up to 720 records)
  - `all_time` — Historical (up to 1000 records)

  Each snapshot contains:
  - `created_at` (number) — Unix timestamp
  - `deposits` (number) — Total deposits
  - `deposits_usd` (number) — Total deposits in USD
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/farms/{address}/stats"
```
</RequestExample>

<ResponseExample>
```json 200 - Success
{
  "status": "ok",
  "stats": {
    "daily": [
      { "created_at": 1709913600, "deposits": 50000.0, "deposits_usd": 17500.0 }
    ],
    "weekly": [],
    "monthly": [],
    "all_time": []
  }
}
```
</ResponseExample>
