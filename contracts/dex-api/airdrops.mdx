## List Airdrops

```
GET https://sei-api.dragonswap.app/api/v1/airdrops
```

Returns all available airdrop campaigns.

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="airdrops" type="array">
  Array of airdrop objects. Each contains:
  - `name` (string) — Airdrop campaign name
  - `address` (string) — Airdrop distributor contract address
  - `token_address` (string) — Airdrop token contract address
  - `number_of_users` (number) — Number of eligible users
  - `timestamps` (array) — Array of vesting unlock timestamps
  - `vesting_text` (string) — Human-readable vesting description
  - `total_claimed` (number) — Total tokens claimed so far
  - `total_deposited` (number) — Total tokens deposited for distribution
  - `starts_at` (number) — Claim start timestamp
  - `ends_at` (number) — Claim end timestamp
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/airdrops"
```

```javascript JavaScript
const response = await fetch("https://sei-api.dragonswap.app/api/v1/airdrops");
const data = await response.json();

console.log(`${data.airdrops.length} airdrop campaigns available`);
```
</RequestExample>

## User Airdrop Allocation

```
GET https://sei-api.dragonswap.app/api/v1/airdrops/{airdrop}/user/{user}
```

Returns a user's airdrop eligibility and allocation amounts for a specific campaign.

### Path Parameters

<ParamField path="airdrop" type="string" required>
  Airdrop contract address
</ParamField>

<ParamField path="user" type="string" required>
  User wallet address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="is_eligible" type="boolean">
  Whether the user is eligible for this airdrop.
</ResponseField>

<ResponseField name="amounts" type="array">
  Array of claimable amounts as strings (one per vesting period, in human-readable token units).
</ResponseField>

<ResponseField name="amounts_wei" type="array">
  Array of claimable amounts in wei as strings (one per vesting period).
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/airdrops/{airdrop}/user/{user}"
```
</RequestExample>

<ResponseExample>
```json 200 - Success
{
  "status": "ok",
  "is_eligible": true,
  "amounts": [
    "123.9",
    "53.1"
  ],
  "amounts_wei": [
    "123900000000000000000",
    "53100000000000000000"
  ]
}
```
</ResponseExample>
