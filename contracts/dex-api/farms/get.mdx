---
title: "Get Farm"
description: "Returns details for a single farm"
api: "GET https://sei-api.dragonswap.app/api/v1/farms/{address}"
---

### Path Parameters

<ParamField path="address" type="string" required default="0x256df69a9a1e7ccd48ffc4211fcaba39a56635a9">
  Farm contract address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="tokens" type="array">
  Tokens associated with this farm.
</ResponseField>

<ResponseField name="farm" type="object">
  Single farm object with the same fields as the list endpoint, plus:
  - `info` (object | null) — Additional farm metadata
</ResponseField>
