---
title: "Farm History"
description: "Transaction history for a farm (deposits, withdrawals, payouts)"
api: "GET https://sei-api.dragonswap.app/api/v1/farms/{address}/history"
---

### Path Parameters

<ParamField path="address" type="string" required default="0x256df69a9a1e7ccd48ffc4211fcaba39a56635a9">
  Farm contract address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="tokens" type="array">
  Tokens associated with this farm (reward and booster tokens).
</ResponseField>

<ResponseField name="history" type="array">
  Array of farm history events. Each contains:
  - `timestamp` (number) — Unix timestamp
  - `amount` (number) — Token amount (for deposits/withdrawals)
  - `user` (string) — User wallet address
  - `type` (string) — Event type: `"deposits"`, `"withdraws"`, or `"payouts"`
  - `pooled_token_address` (string) — Pooled token address
  - `block_number` (number) — Block number
  - `farm_address` (string) — Farm contract address
  - `tx_hash` (string) — Transaction hash

  Payout events additionally include:
  - `reward_token_claim` (number) — Reward tokens claimed
  - `booster_token_claim` (number) — Booster tokens claimed (for boosted farms)
</ResponseField>
