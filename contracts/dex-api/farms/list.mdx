---
title: "List Farms"
description: "Returns all active yield farms with their associated tokens and pools"
api: "GET https://sei-api.dragonswap.app/api/v1/farms"
---

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="tokens" type="array">
  Tokens used across farms. Each contains:
  - `address` (string) — Token contract address
  - `name` (string) — Token name
  - `symbol` (string) — Token symbol
  - `usd_price` (number) — Current USD price
  - `decimals` (number) — Token decimals
</ResponseField>

<ResponseField name="pools" type="array">
  Pools associated with the farms (same shape as the Pools endpoint).
</ResponseField>

<ResponseField name="farms" type="array">
  Array of farm objects. Each contains:
  - `farm_address` (string) — Farm contract address
  - `pooled_token_address` (string) — Staked token address
  - `reward_token_address` (string) — Reward token address
  - `booster_token_address` (string | null) — Booster token address
  - `apy` (number) — Annual percentage yield
  - `tvl` (number) — Total value locked in USD
  - `type` (string) — Farm type: `"basic"` or `"boosted"`
  - `unlock_rate` (number) — Reward per second
  - `booster_unlock_rate` (number | null) — Booster reward per second
  - `is_lp_pooled_token` (boolean) — Whether staked token is an LP token
  - `end_timestamp` (number) — Farm end timestamp
  - `start_timestamp` (number) — Farm start timestamp
  - `total_rewards_usd` (number) — Total rewards value in USD
  - `campaign` (string | null) — Campaign identifier
  - `is_funded` (boolean | null) — Whether the farm is funded
</ResponseField>
