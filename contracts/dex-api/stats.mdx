## Global Stats

```
GET https://sei-api.dragonswap.app/api/v1/stats
```

Returns aggregate statistics for the DEX.

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="stats" type="object">
  Contains a `v2` object with aggregate and 24-hour metrics:
  - `total_volume_usd` (number) — All-time total volume in USD
  - `total_liquidity_usd` (number) — Current total liquidity in USD
  - `total_number_of_swaps` (number) — All-time swap count
  - `total_users_count` (number) — Unique user count
  - `24_hour_tx_count` (number) — Transaction count in the last 24 hours
  - `24_hour_fees_usd` (number) — Fees collected in the last 24 hours (USD)
  - `24_hour_volume_usd` (number) — Volume in the last 24 hours (USD)
  - `hourly_data` (array) — Hourly snapshots, each containing:
    - `created_at` (number) — Unix timestamp
    - `volume_usd` (number) — Hourly volume in USD
    - `total_liquidity_usd` (number) — Liquidity at snapshot time
    - `inflows_usd` (number) — Liquidity inflows in USD
    - `outflows_usd` (number) — Liquidity outflows in USD
    - `fees_usd` (number) — Fees collected in USD
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/stats"
```

```javascript JavaScript
const response = await fetch("https://sei-api.dragonswap.app/api/v1/stats");
const data = await response.json();

const { v2 } = data.stats;
console.log("Total liquidity:", v2.total_liquidity_usd);
console.log("24h volume:", v2["24_hour_volume_usd"]);
```
</RequestExample>

## Transactions

```
GET https://sei-api.dragonswap.app/api/v1/transactions
```

Returns recent transactions with associated token data.

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="tokens" type="array">
  Tokens involved in the transactions. Each contains:
  - `address` (string) — Token contract address
  - `name` (string) — Token name
  - `symbol` (string) — Token symbol
  - `usd_price` (number) — Current USD price
  - `decimals` (number) — Token decimals
</ResponseField>

<ResponseField name="transactions" type="array">
  Array of transaction objects. Each contains:
  - `pool_address` (string) — Pool contract address
  - `first_token_address` (string) — First token in the swap
  - `second_token_address` (string) — Second token in the swap
  - `first_amount_token` (number) — Amount of first token
  - `second_amount_token` (number) — Amount of second token
  - `block_number` (number) — Block number
  - `tx_hash` (string) — Transaction hash
  - `from` (string) — Wallet that initiated the transaction
  - `total_value` (number) — Transaction USD value
  - `type` (string) — `"Swap"`
  - `action` (string) — `"Buy"` or `"Sell"`
  - `created_at` (number) — Unix timestamp
  - `pool_type` (string) — `"v2"` or `"v3"`
</ResponseField>

<RequestExample>

## Volume

```
GET https://sei-api.dragonswap.app/api/v1/vol
```

Returns daily volume statistics for a wallet address, grouped by date.

### Query Parameters

<ParamField query="address" type="string" required>
  Wallet address to query volume for
</ParamField>

### Response

An object keyed by date strings (`YYYY-MM-DD`). Each value contains:

<ResponseField name="volume_usd" type="number">
  Total USD volume for the day
</ResponseField>

<ResponseField name="swap_fee_usd" type="number">
  Total swap fees in USD for the day
</ResponseField>

<ResponseField name="count" type="number">
  Number of swaps for the day
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/vol?address=0x..."
```
</RequestExample>

<ResponseExample>
```json 200 - Success
{
  "2025-12-10": {
    "volume_usd": 402.498,
    "swap_fee_usd": 1.207,
    "count": 5
  },
  "2025-12-23": {
    "volume_usd": 1809.183,
    "swap_fee_usd": 5.276,
    "count": 10
  }
}
```
</ResponseExample>
