---
title: "Pool Stats"
description: "Historical statistics for a pool across multiple time periods"
api: "GET https://sei-api.dragonswap.app/api/v1/pools/{address}/stats"
---

## Path Parameters

<ParamField path="address" type="string" required default="0x4cea29353508b81b6efbfb8d089efe808e8fc991">
  Pool contract address.
</ParamField>

## Query Parameters

<ParamField query="weekly" type="number">
  Number of weekly data points to return. Maximum `100`.
</ParamField>

<ParamField query="monthly" type="number">
  Number of monthly data points to return. Maximum `200`.
</ParamField>

<ParamField query="all_time" type="number">
  Number of all-time data points to return. Maximum `500`.
</ParamField>
