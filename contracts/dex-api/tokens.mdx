## List Tokens

```
GET https://sei-api.dragonswap.app/api/v1/tokens
```

Returns all tokens with liquidity above the minimum TVL threshold.

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="tokens" type="array">
  Array of token objects. Each contains:
  - `address` (string) — Token contract address
  - `name` (string) — Token name
  - `symbol` (string) — Token symbol
  - `usd_price` (number) — Current USD price
  - `decimals` (number) — Token decimals
  - `liquidity` (number) — Total USD liquidity
  - `description` (string | null) — Token description
  - `change` (object) — Price change percentages:
    - `hourly` (number) — Last hour change %
    - `daily` (number) — Last 24h change %
    - `weekly` (number) — Last 7d change %
    - `monthly` (number) — Last 30d change %
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/tokens"
```

```javascript JavaScript
const response = await fetch("https://sei-api.dragonswap.app/api/v1/tokens");
const data = await response.json();

for (const token of data.tokens) {
  console.log(`${token.symbol}: $${token.usd_price} (24h: ${token.change.daily}%)`);
}
```
</RequestExample>

<ResponseExample>
```json 200 - Success
{
  "status": "ok",
  "tokens": [
    {
      "address": "0xe30fedd158a2e3b13e9badaeabafc5516e95e8c7",

    }
  ]
}
```
</ResponseExample>

## Get Token

```
GET https://sei-api.dragonswap.app/api/v1/tokens/{address}
```

Returns details for a single token.

### Path Parameters

<ParamField path="address" type="string" required>
  Token contract address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="token" type="object">
  Single token object containing:
  - `address` (string) — Token contract address
  - `name` (string) — Token name
  - `symbol` (string) — Token symbol
  - `usd_price` (number) — Current USD price
  - `decimals` (number) — Token decimals
  - `liquidity` (number) — Total USD liquidity
  - `daily_volume` (number) — 24-hour USD trading volume
  - `daily_transactions` (number) — Transactions in the last 24 hours

</ResponseField>

## Token Stats

```
GET https://sei-api.dragonswap.app/api/v1/tokens/{address}/stats
```

Returns historical price, liquidity, and volume snapshots for a token.

### Path Parameters

<ParamField path="address" type="string" required>
  Token contract address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="stats" type="object">
  Object containing snapshot arrays for each time period:
  - `daily` — Last 24 hours
  - `weekly` — Last 7 days (up to 168 records)
  - `monthly` — Last 30 days (up to 720 records)
  - `all_time` — Historical (up to 1000 records)

  Each snapshot contains:
  - `usd_price` (number) — USD price at snapshot time
  - `liquidity` (number) — USD liquidity at snapshot time
  - `daily_volume` (number) — USD volume at snapshot time
  - `created_at` (number) — Unix timestamp
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/tokens/{address}/stats"
```
</RequestExample>

<ResponseExample>
```json 200 - Success
{
  "status": "ok",
  "stats": {
    "daily": [
      {
        "usd_price": 0.35,
        "liquidity": 1200000.0,
        "daily_volume": 85000.0,
        "created_at": 1709913600
      }
    ],
    "weekly": [],
    "monthly": [],
    "all_time": []
  }
}
```
</ResponseExample>
