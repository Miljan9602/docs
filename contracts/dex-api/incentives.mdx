## List Incentives

```
GET https://sei-api.dragonswap.app/api/v1/incentives
```

Returns all active and past incentive programs.

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="incentives" type="array">
  Array of incentive objects. Each contains:
  - `start_time` (number) — Incentive start timestamp
  - `end_time` (number) — Incentive end timestamp
  - `reward` (number) — Total reward amount
  - `reward_token` (string) — Reward token contract address
  - `refundee` (string) — Address eligible for refunds
  - `pool` (string) — Associated pool address
  - `incentive_id` (string) — Unique incentive identifier
  - `apr` (number) — Annual percentage rate
  - `title` (string) — Incentive title
  - `liquidity_usd` (number) — Associated liquidity value in USD
  - `owner` (string) — Incentive owner address
  - `campaign` (string) — Campaign identifier
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/incentives"
```

```javascript JavaScript
const response = await fetch("https://sei-api.dragonswap.app/api/v1/incentives");
const data = await response.json();

const active = data.incentives.filter(i => i.end_time > Date.now() / 1000);
console.log(`${active.length} active incentives`);
```
</RequestExample>

## Incentive Tokens

```
GET https://sei-api.dragonswap.app/api/v1/incentives/{incentive_id}/tokens
```

Returns the token IDs associated with a specific incentive program.

### Path Parameters

<ParamField path="incentive_id" type="string" required>
  Unique incentive identifier
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="token_ids" type="array">
  Array of token IDs staked in the incentive program.
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/incentives/{incentive_id}/tokens"
```
</RequestExample>

<ResponseExample>
```json 200 - Success
{
  "status": "ok",
  "token_ids": []
}
```
</ResponseExample>

## User Incentives

```
GET https://sei-api.dragonswap.app/api/v1/incentives/user/{address}
```

Returns incentive program participation and rewards for a specific user.

### Path Parameters

<ParamField path="address" type="string" required>
  User wallet address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="staked_tokens" type="array">
  Tokens the user has staked in incentive programs.
</ResponseField>

<ResponseField name="deposited_tokens" type="array">
  Tokens the user has deposited.
</ResponseField>

<ResponseField name="tokens" type="array">
  Token metadata for the user's positions.
</ResponseField>

<ResponseField name="incentives" type="array">
  Incentive programs the user is participating in.
</ResponseField>

<ResponseField name="pools" type="array">
  Pools associated with the user's incentive positions.
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/incentives/user/{address}"
```
</RequestExample>
