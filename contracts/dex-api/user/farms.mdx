---
title: "User Farms"
description: "All farms where the user has a non-zero balance"
api: "GET https://sei-api.dragonswap.app/api/v1/user/{address}/farms"
---

### Path Parameters

<ParamField path="address" type="string" required>
  User wallet address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="tokens" type="array">
  Tokens associated with the user's farm positions. Each entry includes `address`, `name`, `symbol`, `usd_price`, and `decimals`.
</ResponseField>

<ResponseField name="pools" type="array">
  Pools underlying the user's farmed LP positions.
</ResponseField>

<ResponseField name="farms" type="array">
  Farms where the user has a non-zero balance. Each entry includes standard farm fields such as `farm_address`, `pooled_token_address`, `reward_token_address`, `booster_token_address`, `apy`, `tvl`, `type`, `unlock_rate`, `booster_unlock_rate`, `is_lp_pooled_token`, `end_timestamp`, `start_timestamp`, `total_rewards_usd`, and `campaign`.
</ResponseField>
