---
title: "User Pools"
description: "A user's liquidity pool positions across V2 and V3 pools"
api: "GET https://sei-api.dragonswap.app/api/v1/user/{address}/pools"
---

### Path Parameters

<ParamField path="address" type="string" required>
  User wallet address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="tokens" type="array">
  Tokens involved in the user's pool positions. Each entry includes `address`, `name`, `symbol`, `usd_price`, and `decimals`.
</ResponseField>

<ResponseField name="pools" type="array">
  The user's liquidity pool positions across V2 and V3 pools.
</ResponseField>

<ResponseField name="incentives" type="array">
  Active incentive programs associated with the user's pool positions.
</ResponseField>
