## Staking Info

```
GET https://sei-api.dragonswap.app/api/v1/staking/{address}
```

Returns the next reward emission timestamps for a staking contract.

### Path Parameters

<ParamField path="address" type="string" required>
  Staking contract address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="emission_timestamps" type="array">
  Array of upcoming emission timestamps (next two reward distributions at 18:00 UTC daily).
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/staking/{address}"
```
</RequestExample>

<ResponseExample>
```json 200 - Success
{
  "status": "ok",
  "emission_timestamps": [1709920800, 1710007200]
}
```
</ResponseExample>

## Staking Emissions

```
GET https://sei-api.dragonswap.app/api/v1/staking/{address}/emissions
```

Returns staking deposit records for a staking contract address.

### Path Parameters

<ParamField path="address" type="string" required>
  Staking contract address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="deposits" type="array">
  Array of deposit records sorted by most recent first. Each contains:
  - `tx_hash` (string) — Transaction hash
  - `token_address` (string) — Staked token address
  - `amount` (number) — Staked amount
  - `amount_wei` (string) — Staked amount in wei
  - `created_at` (string) — Unix timestamp as string
  - `apr` (number) — APR at time of deposit
  - `reward_per_usd_staked` (number) — Reward rate per USD staked
  - `token_usd_price` (number) — Token USD price at time of deposit
  - `buy_tx_hash` (string | null) — Associated buy transaction hash
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/staking/{address}/emissions"
```

## Staking Rewards

```
GET https://sei-api.dragonswap.app/api/v1/staking/{address}/rewards
```

Returns protocol rewards, withdrawal fees, and marketplace fees for a staking contract.

### Path Parameters

<ParamField path="address" type="string" required>
  Staking contract address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="fees" type="object">
  Aggregate fee totals:
  - `total_withdrawal_fees_drg` (number) — Total withdrawal fees in DRG
  - `total_marketplace_fees_drg` (number) — Total marketplace fees in DRG
  - `total_protocol_rewards_usd` (number) — Total protocol rewards in USD
</ResponseField>

<ResponseField name="withdrawl_fees" type="array">
  Withdrawal fee records. Each contains:
  - `tx_hash` (string) — Transaction hash
  - `created_at` (string) — Unix timestamp as string
  - `drg_fee` (number) — Fee amount in DRG
  - `reward_per_usd_staked` (string) — Reward rate per USD staked
</ResponseField>

<ResponseField name="marketplace_fees" type="array">
  Marketplace fee records (same shape as withdrawal fees).
</ResponseField>

<ResponseField name="protocol_rewards" type="array">
  Protocol reward distribution records. Each contains:
  - `tx_hash` (string) — Transaction hash
  - `token_address` (string) — Reward token address
  - `amount` (number) — Reward amount
  - `amount_wei` (string) — Reward amount in wei
  - `created_at` (string) — Unix timestamp as string
  - `apr` (number) — APR at time of distribution
  - `reward_per_usd_staked` (number) — Reward rate per USD staked
  - `token_usd_price` (number) — Token USD price at time of distribution
  - `buy_tx_hash` (string | null) — Associated buy transaction hash
</ResponseField>

## Staking Stats

```
GET https://sei-api.dragonswap.app/api/v1/staking/{address}/stats
```

Returns historical staking statistics across multiple time periods.

### Path Parameters

<ParamField path="address" type="string" required>
  Staking contract address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="total_staked" type="number">
  Current total staked amount
</ResponseField>

<ResponseField name="stats" type="object">
  Object containing snapshot arrays:
  - `daily` — Last 24 hours
  - `weekly` — Last 7 days (up to 168 records)
  - `monthly` — Last 30 days (up to 720 records)
  - `all_time` — Historical (up to 1000 records)

  Each snapshot contains:
  - `total_deposits` (number) — Total deposit amount
  - `total_deposits_usd` (number) — Total deposits in USD
  - `total_fees` (number) — Total fees collected
  - `apr` (number) — APR at snapshot time
  - `created_at` (number) — Unix timestamp
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/staking/{address}/stats"
```
</RequestExample>

<ResponseExample>
```json 200 - Success
{
  "status": "ok",
  "total_staked": 25182898.98,
  "stats": {
    "daily": [
      {
        "total_deposits": 25176799.274119,
        "total_deposits_usd": 425155.73,
        "total_fees": 13126.065,
        "apr": 235.207,
        "created_at": 1772971810
      }
    ],

    "monthly": []
  }
}
```

## Staking Activity

```
GET https://sei-api.dragonswap.app/api/v1/staking/{address}/activity/{user}
```

Returns a user's staking activity, including rewards, staking events, marketplace actions, and bonuses.

### Path Parameters

<ParamField path="address" type="string" required>
  Staking contract address
</ParamField>

<ParamField path="user" type="string" required>
  User wallet address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="data" type="object">
  Object containing four activity arrays:

  **`reward_activity`** — Reward claim events with `type`, `user`, `token_operation`, `claimed_amount_wei`, `drg_amount_wei`, `usdc_amount_wei`, `token`, `tx_hash`, `timestamp`

  **`staking_activity`** — Stake/unstake events with `type`, `user`, `amount_wei`, `fees`, `tx_hash`, `timestamp`

  **`marketplace_activity`** — Marketplace events with `type`, `user`, `amount_wei`, `price_in_USDC_wei`, `sold_to`, `bought_from`, `tx_hash`, `timestamp`

  **`staking_bonus`** — Bonus rewards with `type` (`"Bonus"`), `user`, `token`, `amount`, `amount_wei`, `amount_usd_value`, `tx_hash`, `timestamp`
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/staking/{address}/activity/{user}"
```
</RequestExample>

## User Staking Stats

```
GET https://sei-api.dragonswap.app/api/v1/staking/{address}/stats/{user}
```

Returns aggregated staking V2 statistics for a specific user.

### Path Parameters

<ParamField path="address" type="string" required>
  Staking contract address
</ParamField>

<ParamField path="user" type="string" required>
  User wallet address
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"ok"` on success
</ResponseField>

<ResponseField name="stats" type="object | null">
  User staking stats (null if user has no staking activity). When present, contains:
  - `total_DRG_earned` (number) — Total DRG tokens earned
  - `total_USDC_earned` (number) — Total USDC earned
  - `total_staked` (number) — Total amount currently staked
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://sei-api.dragonswap.app/api/v1/staking/{address}/stats/{user}"
```
</RequestExample>
